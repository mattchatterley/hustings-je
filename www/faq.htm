Frequently Asked Questions

How does it work?

We have a Hadoop cluster in AWS that is collecting and analysing the tweets, then pushing them out to the website.
All the source code is here [link] if you want to see for yourself.

How do you analyse sentiment?

There are two methods: simple (or “bag of words”) and NLP.
The simple methods score individual words as either positive, neutral or negative, and the score for a tweet is simply added to give an overall score. NLP stands for Natural Language Processing. We use the Stanford University NLP Group’s CoreNLP suite of tools (http://nlp.stanford.edu/software/corenlp.shtml). This uses a neural network trained on movie reviews to generate the sentiment scores. This is currently the most accurate machine-learning algorithm for sentiment scoring.

I've found one you got wrong??!?!

Yeah, that’ll happen.
Neither simple nor NLP scoring is a 100% accurate process. Stanford have claimed an 85% accuracy for their sentiment analyser. We will be testing the accuracy of it in this context.

You’ve made it biased against/for XYZ haven’t you?

No, we really haven’t.
We’re good, but not good enough to introduce subtle-yet-influential-but-impossible-to-find biases. All the source code is here [link] so you can verify it all for yourself. Let us know if you want a hand.
